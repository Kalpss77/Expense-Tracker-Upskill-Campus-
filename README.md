# Expense Tracker Application

## Description
This is a console-based Expense Tracker developed using Core Java.
It helps users record and manage daily expenses.

## Features
- Add new expense
- View all expenses
- Search expenses by category
- Calculate total expense
- Delete expense

## Technologies Used
- Core Java
- ArrayList
- Scanner class
